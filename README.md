# Networking & security project
A Secured Small Enterprise Network (cisco packet tracer)
# project Overview
**Welcome to the *"networking & security project"* this project was developed using cisco packet tracer ,showcasing a comprehensive network setup. the architecture comprises of 1 router ,1 switch , 6PCs , 2 laptops and a server . this network is secured by network segementation and password security on devices**
# key components
*      VLAN 10-IT > 192.168.10.0/24(1 PC, SERVER)
*     VLAN 20-MANAGEMENT > 192.168.20.0/24(2PCs)
*     VLAN 30-STAFF > 192.168.30.0/24(2 PCs, 2 LAPTOPS)
*     EXTERNAL NETWORK (INTERNET SIMULATION) > 10.10.10.0/24(1 PCs)
*     1 ROUTER
*     1 SWITCH
